### 获取并保存Cookies的代码

以下代码段可以获取当前网站的所有Cookies并将它们保存为一个JSON对象：

```javascript
(function() {
    var cookies = document.cookie.split(";").map(function(cookie) {
        var parts = cookie.split("=");
        return {name: parts[0].trim(), value: parts.slice(1).join("=")};
    });
    console.log(JSON.stringify(cookies));
})();
```

### 设置Cookies的代码

(function() {
